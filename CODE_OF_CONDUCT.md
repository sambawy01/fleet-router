# Code of Conduct

## Our standards

- Be respectful and constructive in all interactions.
- Accept criticism gracefully. Disagree on ideas, not people.
- Focus on what's best for the project and its users.

## Unacceptable behavior

- Harassment, discrimination, or intimidation of any kind.
- Trolling, insulting/derogatory comments, or personal attacks.
- Publishing others' private information without permission.

## Reporting

If you experience or witness unacceptable behavior, email [maintainer contact] or open a private issue. All reports will be handled confidentially.

## Scope

This code applies to all project spaces: GitHub issues, PRs, discussions, and any community channels.
